# WeLink 前端开发文档

## 1. 概述

本文档描述 WeLink 即时通讯系统前端部分的开发规范、架构设计和技术实现。前端采用 Vue 3 + Element Plus 技术栈，通过 REST API 和 WebSocket 与后端进行通信。

**开发日期：** 2026-05-07

**技术栈：**
- Vue 3.4 - 前端框架
- Element Plus 2.6 - UI 组件库
- Pinia 2.1 - 状态管理
- Vue Router 4.3 - 路由管理
- Axios 1.6 - HTTP 客户端
- Vite 5.2 - 构建工具
- WebSocket - 实时通信

## 4. 核心模块说明

### 4.1 API 服务层 (`src/api/index.js`)

封装所有后端 API 接口调用，按功能模块分类：

**认证接口 (`authApi`)**
| 方法 | 路径 | 参数 | 说明 |
|------|------|------|------|
| `register(data)` | POST /auth/register | `{ username, password }` | 用户注册 |
| `login(data)` | POST /auth/login | `{ username, password }` | 用户登录 |
| `getUserInfo(userId)` | GET /user/{userId} | - | 获取用户信息 |

**好友接口 (`friendApi`)**
| 方法 | 路径 | 参数 | 说明 |
|------|------|------|------|
| `sendRequest(friendId)` | POST /friend/apply/{friendId} | - | 发送好友申请 |
| `acceptRequest(friendId)` | POST /friend/accept/{friendId} | - | 接受好友申请 |
| `rejectRequest(friendId)` | POST /friend/reject/{friendId} | - | 拒绝好友申请 |
| `getList()` | GET /friend/list | - | 获取好友列表 |
| `deleteFriend(friendId)` | DELETE /friend/{friendId} | - | 删除好友 |

**群组接口 (`groupApi`)**
| 方法 | 路径 | 参数 | 说明 |
|------|------|------|------|
| `create(data)` | POST /group | `{ groupName, notice }` | 创建群组 |
| `getList()` | GET /group/list | - | 获取群组列表 |
| `getMembers(groupId)` | GET /group/{groupId}/members | - | 获取群成员 |
| `inviteMembers(groupId, memberIds)` | POST /group/{groupId}/invite | `[userId]` | 邀请成员 |
| `kickMember(groupId, targetId)` | DELETE /group/{groupId}/kick/{targetId} | - | 踢出成员 |
| `quitGroup(groupId)` | DELETE /group/{groupId}/quit | - | 退出群组 |

**消息接口 (`messageApi`)**
| 方法 | 路径 | 参数 | 说明 |
|------|------|------|------|
| `getPrivateHistory(params)` | GET /message/history/private | `{ targetUserId, page, pageSize }` | 私聊历史 |
| `getGroupHistory(params)` | GET /message/history/group | `{ groupId, page, pageSize }` | 群聊历史 |
| `getOfflineMessages()` | GET /message/offline | - | 离线消息 |
| `markAsRead(msgId)` | POST /message/read/{msgId} | - | 标记已读 |

**文件接口 (`fileApi`)**
| 方法 | 路径 | 参数 | 说明 |
|------|------|------|------|
| `upload(file)` | POST /file/upload | `File` | 文件上传 |

### 4.2 WebSocket 服务 (`src/utils/websocket.js`)

封装 WebSocket 连接管理，提供以下功能：

**连接管理**
- 自动重连机制（最多 5 次）
- 心跳检测（每 30 秒）
- 连接状态监控

**消息协议**

```javascript
// 认证消息
{
  "type": "auth",
  "token": "<jwt_token>"
}

// 发送消息（私聊）
{
  "type": "message",
  "toUserId": 123,
  "msgType": 1,
  "content": "你好"
}

// 发送消息（群聊）
{
  "type": "message",
  "groupId": 456,
  "msgType": 1,
  "content": "大家好"
}

// 心跳消息
{
  "type": "heartbeat"
}

// 确认消息
{
  "type": "ack",
  "msgId": "uuid"
}

// 系统通知
{
  "type": "system",
  "action": "online/offline",
  "userId": 123
}
```

**使用方法**

```javascript
import { wsService } from '@/utils/websocket'

// 连接
wsService.connect(token)

// 监听消息
wsService.onMessage((data) => {
  if (data.type === 'message') {
    // 处理聊天消息
  } else if (data.type === 'system') {
    // 处理系统通知
  }
})

// 发送消息
wsService.sendMessage({
  toUserId: 123,
  msgType: 1,
  content: '你好'
})

// 断开连接
wsService.disconnect()
```

### 4.3 状态管理 (`src/stores/`)

使用 Pinia 进行全局状态管理，分为三个核心 store：

**用户状态 (`user.js`)**
```javascript
{
  token: '',           // JWT Token
  userInfo: {          // 用户信息
    id: 123,
    username: 'user123',
    nickname: '用户',
    avatar: '...'
  }
}
```

**联系人状态 (`contact.js`)**
```javascript
{
  friends: [],         // 好友列表
  groups: [],          // 群组列表
  friendRequests: []   // 好友申请
}
```

**聊天状态 (`chat.js`)**
```javascript
{
  conversations: [],       // 会话列表
  currentConversation: {}, // 当前会话
  messages: [],            // 消息列表
  unreadCount: 0           // 未读数
}
```

## 7. 编码规范

### 7.1 命名规范

**文件命名：**
- 组件文件：PascalCase（如 `ChatWindow.vue`）
- 工具文件：camelCase（如 `websocket.js`）
- 样式文件：kebab-case（如 `chat-window.scss`）

**变量命名：**
- 变量/函数：camelCase
- 常量：UPPER_SNAKE_CASE
- 组件名：PascalCase

### 7.2 代码风格

- 使用 Composition API (`<script setup>`)
- 优先使用响应式 API（`ref`, `reactive`）
- 组件 props 需要类型定义
- 使用 ESLint + Prettier 格式化代码

### 7.3 组件结构

```vue
<template>
  <!-- 模板内容 -->
</template>

<script setup>
import { ref, computed, onMounted } from 'vue'

// Props 定义
const props = defineProps({
  title: {
    type: String,
    required: true
  }
})

// Emits 定义
const emit = defineEmits(['update', 'delete'])

// 响应式数据
const loading = ref(false)

// 计算属性
const isValid = computed(() => {
  return props.title.length > 0
})

// 方法
const handleSubmit = async () => {
  loading.value = true
  try {
    // 业务逻辑
  } finally {
    loading.value = false
  }
}

// 生命周期
onMounted(() => {
  // 初始化逻辑
})
</script>

<style scoped>

</style>
```

## 8. 部署说明

### 8.1 Nginx 配置

```nginx
server {
    listen 80;
    server_name welink.example.com;

    # 前端静态文件
    location / {
        root /usr/share/nginx/html;
        index index.html;
        try_files $uri $uri/ /index.html;
    }

    # API 代理
    location /api/ {
        proxy_pass http://backend:8080;
        proxy_set_header Host $host;
        proxy_set_header X-Real-IP $remote_addr;
    }

    # WebSocket 代理
    location /ws {
        proxy_pass http://backend:8081;
        proxy_http_version 1.1;
        proxy_set_header Upgrade $http_upgrade;
        proxy_set_header Connection "upgrade";
        proxy_set_header Host $host;
    }
}
```

### 8.2 Docker 部署

```dockerfile
FROM node:18-alpine AS build
WORKDIR /app
COPY package*.json ./
RUN npm install
COPY . .
RUN npm run build

FROM nginx:alpine
COPY --from=build /app/dist /usr/share/nginx/html
COPY nginx.conf /etc/nginx/conf.d/default.conf
EXPOSE 80
CMD ["nginx", "-g", "daemon off;"]
```

## 9. 常见问题

### 9.1 WebSocket 连接失败

**问题：** WebSocket 连接被拒绝

**解决方案：**
1. 检查后端 WebSocket 服务是否启动（默认端口 8081）
2. 检查防火墙设置
3. 确认 WebSocket URL 配置正确

### 9.2 Token 过期处理

**问题：** 用户操作时 Token 过期

**解决方案：**
1. Axios 拦截器自动检测 401 状态码
2. 清除本地存储并跳转到登录页
3. 提示用户重新登录

### 9.3 消息不显示

**问题：** 发送消息后不显示在聊天窗口

**解决方案：**
1. 检查 WebSocket 连接状态
2. 确认消息格式正确
3. 查看浏览器控制台错误信息
